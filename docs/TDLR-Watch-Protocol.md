# TDLR Watch Protocol — Layer 1A (v1.7)

## Overview
Any pre-1978 structure requires a TABS asbestos abatement notification
filed with TDLR before demolition begins. This filing reveals the GC.

## Search Instructions
URL: https://tdlr.texas.gov/tabs/search.htm

Search fields (run in order):
  A. Property address
  B. Owner name
  C. Zip code + recent filter

## Status Classifications
| Status | Condition | Action |
|--------|-----------|--------|
| CONFIRMED | TABS filing found with GC name | Sub-contract pursuit |
| PENDING | Approval exists, no TABS yet | Monitor weekly |
| NOT REQUIRED | Post-1978 structure | Skip to Accela only |
| UNKNOWN | Era unconfirmed | Treat as pre-1978 |

## GC Contact Script
"I saw you pulled the abatement notification on [property].
I do salvage sub-work — I handle material extraction at no cost
to your crew, sometimes with a revenue share. Are you already
locked in on how you're clearing that site?"
